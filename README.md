# Jerome Bizimana — Portfolio

Personal portfolio site built with Jekyll using the Minimal Mistakes theme.

## Local development

1. Install dependencies:
	- `bundle install`
2. Run the site:
	- `bundle exec jekyll serve`
3. Open:
	- http://127.0.0.1:4000

## Key files

- `_config.yml` — site title, bio, social links
- `_data/navigation.yml` — top navigation
- `index.html` — homepage content
- `_pages/` — About, Contact, etc.
